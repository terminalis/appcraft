Diff, not drift.

AppCraft is the deterministic app compiler for the AI era. An AI agent writes a small YAML model — never code. The compiler emits a complete native Android app you fully own. A feature request is a 2-line reviewable diff.

The change request is the diff

$ git diff app.acm.yaml
   attributes:
     - { name: mmol,    type: decimal }
+    - { name: fasting, type: bool }
   flows:
     - name: history
-      sort: { by: takenAt, order: desc }
+      sort: { by: takenAt, order: asc }
What regenerates from those 2 lines:

  Room schema & entity
  the create form (new toggle)
  the list card & sort
  the data facade

Deterministically. Same model +
same compiler version → byte-identical
output. No LLM below the model.

Install

For AI agents (the primary door) — one command:

claude mcp add appcraft -- npx -y @appcraft-io/mcp-server

For developers and CI:

npx appcraft validate app.acm.yaml
npx appcraft generate app.acm.yaml -o my-app-android
npx appcraft preview  app.acm.yaml -o preview.html
npx appcraft schema --card

Why a compiler beats codegen

Guarantees
The compiler proves properties of its output: architecture invariants, correct permission manifests, lint-clean templates. An LLM can only probably do these.
Auditability
A model diff is reviewable by a non-engineer, an auditor, or an agent verifying its own work. A 20,000-line codebase diff is not.
Re-targetability
The same model recompiles to next year's toolchain or a new platform. No vibe-coded app migrates itself.

Honest about the ceiling

AppCraft compiles data + flows + charts companion apps: trackers, clinical companions, field tools, internal tools — with typed custom: Kotlin escape hatches preserved verbatim across regeneration. It is deliberately not for games, social feeds, or video editors. What it can and cannot do is a machine-readable capability card agents must consult — the card never lies. Every example builds in CI against the real Android toolchain (Compose, Material 3, Room) behind a zero-open-issues SonarQube Cloud gate.
