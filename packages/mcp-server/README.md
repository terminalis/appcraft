# @appcraft-io/mcp-server

MCP server for the AppCraft deterministic app compiler — agents build native Android apps
via `get_schema`, `create_app`, `edit_model`, `validate`, `compile`, `preview`.

```bash
claude mcp add appcraft -- npx -y @appcraft-io/mcp-server
```

Part of [AppCraft](https://github.com/terminalis/appcraft) — the deterministic app compiler for the AI era.
