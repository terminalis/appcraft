# appcraft

Deterministic app compiler for the AI era — CLI. Validate, generate, and preview complete
native Android apps from a small YAML model.

```bash
npx appcraft validate app.acm.yaml
npx appcraft generate app.acm.yaml -o my-app-android
```

Part of [AppCraft](https://github.com/terminalis/appcraft) — the deterministic app compiler for the AI era.
